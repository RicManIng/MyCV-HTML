Progetti

  • Controllo banco test durata HW - Zapi spa

    Strumenti utilizzati :

    • microcontrollore Microchip - PIC32MX
    • programmazione in C
    • scheda demo PIC32-MAXI-WEB
    • MPLABXIde
    • CANAnalyzer Ixxat
    • FreeRTOS

    Descrizione :

    Il progetto sviluppato effettua un pilotaggio del dispositivo sotto test e un controllo sulle variabili d'ambiente in cui la prova è eseguita.
    Nel caso in cui venga rilevata un'anomalia di funzionamento il controllo porta il test in condizioni di sicurezza fino all'intervento dell'operatore.
    Il controllo è stato prodotto scrivendo il codice in C embedded per una scheda demo con a bordo un microcontrollore Microchip parallelizzando i processi utilizzando lo scheduler messo a disposizione dalla libreria FreeRTOS

  • Sviluppo HW/SW banco HIL per test SW - Zapi spa

    Strumenti utilizzati :

    • Raspberry
    • programmazione in Python
    • Periferiche in I2C, SPI, CAN
    • CANAnalyzer Ixxat
    • librerie threading e multiprocessing
    • programmazione ad oggetti

    Descrizione :

    Il progetto sviluppato genera le condizioni di allarme in modo elettrico sollecitando esternamente il componente su cui è installato il SW sotto test.
    Per eseguire queste prove è stato sviluppato un codice in Python orientato agli oggetti e per ottimizzare le prestazioni si sono utilizzate le librerie theading e multiprocessing così da ottenere un tempo di risposta del sistema testante molto più piccolo rispetto al sistema testato.

  • Sviluppo libreria Computer Vision - K-loops srl

    Strumenti utilizzati :

    • Robot 6 assi industriale KUKA
    • programmazione in KRL
    • programmazione in Python
    • IntelRealSense depth camera
    • OpenCV e Open3D

    Descrizione :

    Il progetto riguarda la movimentazione robot mediante l'acquisizione di punti nello spazio tramite una Depth Camera.
    Utilizzando le librerie Open3D e OpenCV si va a cercare nello spazio una nuvola di punti compatibile con un oggetto noto.
    Successivamente si calcola tramite matrici la distanza e l'orientamento dell'oggetto e si pilota il robot con il linguaggio KRL per afferrare l'oggetto.

  • Sviluppo SW controllo macchina automatica - ElettroSoftware srl

    Strumenti utilizzati :

    • PLC testo strutturato
    • Automation Studio B&R

    Descrizione :

    Tramite i componenti e il software offerti da B&R (AutomationStudio) si è sviluppato il codice di controllo di una macchina spalmatriche che regola l'apertura di due lame per aumentare o ridurre la caduta di smalto sulle piastrelle.

  • Sviluppo codice domotica - Università

    Strumenti utilizzati :

    • microcontrollore STM32
    • programmazione in C
    • periferiche I2C, SPI, I2S, Bluetooth

    Descrizione :

    Il progetto sviluppato effettua il controllo e il pilotaggio di alcune periferiche per il funzionamento di impianti di domotica.
    In particolare è stato scritto un codice in C embedded su piattaforma STM32 per :
    • Controllo altezza serrande in base alla luminosità
    • Controllo remoto del livello di luminosità
    • Controllo remoto del condizionamento ambientale
    • Servizio webmail con messaggio vocale in caso di non risposta al campanello
    • Applicazione per Smartphone per la scelta da parte dell'utente dei parametri di configurazione
